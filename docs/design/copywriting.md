# Copywriting Guide

Version: 1.0

---

# 목적

Copywriting Guide는 창제몬 웹앱에서 사용하는 모든 문장의 톤앤매너를 정의한다.

사용자는 앱을 사용하는 동안
따뜻함, 희망, 성장의 감정을 느껴야 한다.

모든 문장은 창제몬의 세계관과 일관된 말투를 유지한다.

---

# Tone & Voice

창제몬은

- 따뜻하다.
- 긍정적이다.
- 부담스럽지 않다.
- 응원하지만 강요하지 않는다.
- 편안한 친구처럼 말한다.

---

# 말투 규칙

사용

- 함께
- 천천히
- 조금씩
- 괜찮아
- 오늘도
- 성장

지양

- 반드시
- 무조건
- 실패
- 틀렸다
- 부족하다
- 잘못했다

---

# 버튼 문구

테스트 시작하기

다시 테스트

공유하기

홈으로

도감 보기

---

# Loading

당신의 성장 성향을 분석하고 있습니다...

잠시만 기다려 주세요.

---

# Result

새로운 창제몬을 만났어요!

도감 등록 완료!

당신의 성장 파트너가 탄생했습니다.

---

# Collection

창제몬 도감

새로운 창제몬을 만나보세요.

아직 만나지 못한 창제몬입니다.

---

# Empty State

아직 만난 창제몬이 없어요.

테스트를 시작하면
첫 번째 창제몬을 만날 수 있어요.

---

# Toast Message

준비 중인 기능입니다.

다음 만남을 기대해 주세요.

도감이 업데이트되었습니다.

---

# Error Message

문제가 발생했어요.

잠시 후 다시 시도해 주세요.

---

# Success Message

완료되었습니다.

도감에 등록되었어요!

---

# Character Dialogue

창제몬의 대사는
별도의 문서에서 관리한다.

copywriting.md에서는
앱 공통 문구만 관리한다.

---

# UX 원칙

문장은 짧고 이해하기 쉽게 작성한다.

부정적인 표현보다
긍정적인 표현을 사용한다.

사용자를 평가하지 않는다.

명령보다 제안하는 표현을 사용한다.

---

# 금지 표현

- 실패했습니다.
- 틀렸습니다.
- 다시 하세요.
- 잘못되었습니다.
- 부족합니다.
- 반드시 해야 합니다.
