# Art Guide

Version: 0.1

---

# 목적

이 문서는 창제몬의 모든 아트 스타일을 정의하는 공식 가이드이다.

모든 이미지 생성, 픽셀아트 제작, 일러스트 제작, 애니메이션 제작은 이 문서를 기준으로 한다.

---

# Golden Rules

창제몬은 항상 귀여워야 한다.

창제몬은 공격적으로 보이면 안 된다.

창제몬은 어린아이도 좋아할 수 있는 디자인이어야 한다.

복잡함보다 직관성을 우선한다.

강함보다 따뜻함을 우선한다.

실루엣만 봐도 누구인지 구분할 수 있어야 한다.

모든 창제몬은 친구처럼 느껴져야 한다.

---

# Art Style

창제몬은 Game Boy Advance Pokémon Generation 3 스타일에서 영감을 받는다.

기본 스타일

- Pixel Art
- Pokémon Gen3 Style
- Cute
- Soft
- Clean
- Nintendo Style

---

# Canvas

기본

48 × 48 px

확장 일러스트

512 × 512 px

공유 이미지

1080 × 1080 px

---

# Pixel Rules

- 16색 이하 팔레트 사용
- 1px 외곽선
- 투명 배경
- 과도한 디더링 사용 금지
- 그림자는 최소한으로 표현
- 선명한 실루엣 유지

---

# Camera

기본 시점

3/4 View

사용자가 캐릭터의 얼굴과 몸을 동시에 볼 수 있도록 한다.

# Face Style

- 둥근 얼굴
- 큰 눈
- 작은 코
- 작은 입
- 항상 부드러운 표정

---

# Expression

기본 표정

😊

사용 가능한 표정

- 웃음
- 놀람
- 졸림
- 기쁨
- 부끄러움

사용하지 않는 표정

- 분노
- 공포
- 위협
- 공격적인 표정

---

# Outline

1px

짙은 갈색 또는 짙은 회색 사용

순수 검정(#000000)은 사용하지 않는다.

---

# Lighting

빛은 좌측 상단에서 들어온다.

모든 캐릭터는 동일한 광원 방향을 유지한다.

---

# Shadow

부드러운 음영만 사용한다.

명암 단계는 최대 3단계까지 사용한다.

---

# Background

기본

Transparent

배경이 필요한 경우

- 숲
- 들판
- 하늘

등 자연 요소를 사용한다.

# Idle Animation

모든 창제몬은 살아있는 것처럼 움직인다.

예시

- 눈 깜빡임
- 귀 움직임
- 꼬리 흔들기
- 살짝 점프
- 몸 흔들기
- 반짝임

---

# Evolution

모든 창제몬은

알

↓

1단계

↓

2단계

↓

최종 진화

의 구조를 가진다.

진화할수록

- 장식이 늘어난다.
- 상징 요소가 강화된다.
- 크기가 약간 커진다.

귀여운 이미지는 유지한다.

---

# Forbidden

다음 요소는 사용하지 않는다.

- 현실적인 털 표현
- 과도한 근육
- 날카로운 이빨
- 피
- 무기
- 갑옷
- 복잡한 장식
- 사람 같은 비율
- 사실적인 그림체
- 과한 명암

---

# Animation Style

Animation은

- Bounce
- Float
- Blink
- Tail Swing
- Ear Move

위주로 사용한다.

모든 움직임은 느리고 부드럽게 표현한다.

# Official Prompt

Create an original cute monster in the official Changjemon art style.

Style:
Game Boy Advance Pokémon Generation 3 sprite style.

Canvas:
48×48 pixels.

Palette:
Maximum 16 colors.

Background:
Transparent.

View:
3/4 perspective.

Outline:
1px dark outline.

Character:
Round body,
large eyes,
short legs,
cute proportions.

Do not use realistic rendering,
complex shadows,
weapons,
or aggressive expressions.

# Quality Checklist

이미지 생성 후 아래 항목을 확인한다.

□ 창제몬처럼 보이는가

□ 귀여운가

□ 실루엣만으로 구분 가능한가

□ 대표 상징이 잘 보이는가

□ 색상이 명확한가

□ 16색 이하인가

□ 3/4 시점인가

□ 공격적으로 보이지 않는가

□ 모바일에서도 잘 보이는가
