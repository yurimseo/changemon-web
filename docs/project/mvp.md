# MVP (Minimum Viable Product)

Version: 1.0

---

# 목표

이번 버전은 행사 체험용 모바일 웹앱을 제작하는 것을 목표로 한다.

사용자는 QR 또는 NFC를 통해 웹앱에 접속하여 약 2~3분 안에 테스트를 완료하고 자신의 창제몬을 만난다.

복잡한 기능보다 완성도 높은 사용자 경험을 우선한다.

# 성공 기준

사용자가

- 테스트를 완료한다.
- 자신의 창제몬을 만난다.
- 결과를 공유한다.
- 홈으로 돌아간다.

위 흐름이 자연스럽게 이어지면 MVP 성공으로 본다.
