# 🌱 Monsters

Version: 1.0

---

# 개요

창제몬은 사용자의 성장 성향을 상징하는 다섯 명의 성장 파트너이다.

모든 창제몬은 각각 하나의 핵심 가치를 품고 태어나며,
사용자의 선택과 실천을 통해 함께 성장한다.

각 창제몬은 서로 우열이 없으며,
모든 성향은 소중한 성장의 모습이다.

# 공통 규칙

모든 창제몬은

- 사용자의 성장 파트너이다.
- 사용자를 평가하지 않는다.
- 함께 성장한다.
- 성장을 응원한다.
- 각각 고유한 진화 형태를 가진다.
- 성향에는 우열이 존재하지 않는다.

---

# 개발 메모

- 모든 캐릭터는 48×48 GBA Pokémon Gen3 스타일 픽셀아트 기준으로 제작한다.
- 16색 이하 팔레트를 사용한다.
- 둥글고 귀여운 비율을 유지한다.
- 각 캐릭터는 알 → 기본형 → 진화형 → 최종진화형의 성장 단계를 가진다.
