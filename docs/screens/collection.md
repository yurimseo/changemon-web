# Collection Screen

Version: 1.0

---

# 목적

Collection Screen은 사용자가 획득한 창제몬을 확인하는 도감 화면이다.

단순한 리스트가 아니라,
창제몬과의 만남을 기록하는 공간이다.

MVP에서는 행사 체험용 웹앱이므로
사용자는 한 번의 테스트를 통해 하나의 창제몬만 획득한다.

도감은 앞으로 확장될 세계를 보여주는 역할을 한다.

---

# MVP Scope

이번 버전에서 구현하는 기능

- 총 5마리 창제몬 표시
- 획득한 창제몬 1마리만 컬러 표시
- 나머지 4마리는 잠금 상태
- 획득한 창제몬만 상세 페이지 진입 가능
- 잠긴 창제몬은 클릭 시 안내 메시지 출력

# Locked Monster

잠긴 창제몬은

???

# Interaction

## 획득한 창제몬

카드 클릭

↓

Monster Detail Screen 이동

---

## 잠긴 창제몬

카드 클릭

↓

Toast Message

"아직 만나지 못한 창제몬이에요."

또는

"다음 만남을 기대해주세요."

상세 화면은 열리지 않는다.

# UX 원칙

- 획득한 창제몬을 가장 먼저 보여준다.
- 잠긴 창제몬은 궁금증을 유발해야 한다.
- 과도한 기능을 넣지 않는다.
- 한눈에 이해할 수 있는 화면을 만든다.

---

# 개발 규칙

- 모바일 전용
- 2열 Grid Layout
- 동일한 카드 크기
- Lazy Loading 불필요 (5장 고정)
- 스크롤 최소화

---

# MVP 개발 규칙

이번 행사 버전에서는

- 사용자는 하나의 창제몬만 획득한다.
- 도감은 수집 기능보다 세계관을 보여주는 역할을 한다.
- 잠긴 창제몬은 데이터만 준비하고 실제 획득 기능은 구현하지 않는다.
