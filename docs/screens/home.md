# Home Screen

Version: 2.0

---

# 목적

Home Screen은 사용자의 창제몬 성장 상태를 한눈에 보여주는 메인 화면이다.

사용자는 현재 창제몬이 어떤 상태인지 확인하고,

다음에 무엇을 해야 하는지 자연스럽게 알 수 있어야 한다.

Home은 언제나 사용자가 가장 먼저 보게 되는 화면이다.

---

# UX 목표

사용자가

"지금 내 창제몬은 어디까지 성장했지?"

를 바로 이해할 수 있도록 한다.

현재 진행 상황을 시각적으로 표현하고,

다음 행동을 자연스럽게 유도한다.

---

# Home State

Home Screen은 하나의 화면을 사용한다.

사용자의 진행 상황에 따라

- 알
- 문구
- 체크리스트
- 버튼

이 자동으로 변경된다.

---

# Background

배경은 성장 상태에 따라 변경된다.

기본 알

연한 크림색

↓

성향 결정

대표 성향 색상이 은은하게 추가된다.

↓

부화 준비

빛나는 효과

↓

창제몬 탄생

숲 배경 + 작은 파티클
