# Monster Detail Screen

Version: 1.0

---

# 목적

Monster Detail Screen은 창제몬 한 마리에 대한 정보를 확인하는 도감 페이지이다.

단순한 정보 화면이 아니라,

사용자가 자신의 창제몬을 이해하고 애정을 느끼는 공간을 목표로 한다.

MVP에서는 획득한 창제몬만 상세 페이지를 볼 수 있다.

---

# UX 목표

사용자가

"내 창제몬 귀엽다."

"나랑 잘 어울리는 것 같다."

라는 감정을 느끼도록 한다.

# 대표 문구

창제몬을 대표하는 한 문장

예시

---

# Encyclopedia

도감 설명

2~3줄 정도의 짧은 소개

예시

리프온은 작은 성장을 소중하게 생각하는 창제몬이다.

누군가의 노력과 도전을 응원하며,
천천히 함께 성장하는 것을 가장 큰 행복으로 여긴다.

# Locked Monster

잠긴 창제몬은 상세 화면이 존재하지 않는다.

카드를 선택하면

"아직 만나지 못한 창제몬입니다."

Toast Message만 출력한다.

---

# MVP Scope

이번 버전에서는

- 획득한 창제몬만 상세 보기 가능
- 진화 정보 없음
- 성장 레벨 없음
- 친밀도 없음
- 스토리 이벤트 없음

---

# UX 원칙

- 정보보다 캐릭터가 먼저 보인다.
- 스크롤을 많이 하지 않는다.
- 도감 한 페이지를 읽는 느낌을 제공한다.
- 사용자가 창제몬에게 애정을 느낄 수 있도록 만든다.

---

# 개발 규칙

- 모바일 전용
- Hero 영역은 첫 화면에서 모두 보이도록 한다.
- 캐릭터 Idle Animation은 항상 실행한다.
- 모든 정보는 하나의 카드 안에 자연스럽게 배치한다.
